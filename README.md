# Interactive Derivative Visualizer

This Python script uses Matplotlib to show functions and their derivatives with smooth animation and interactive controls.

## How to run

1. Download or clone this repo.  
2. Install required packages:
3. Run the script:


## Features

- Animated function and derivative plot  
- Tangent line with moving point  
- Dark/light mode toggle  
- Controls for speed, domain, resolution, and trail length  
- Keyboard shortcuts (space to play/pause, arrows to step)
